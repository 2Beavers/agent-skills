# Project instructions

- If `.agent/resume.md` exists, read it before making major changes.
- Treat repository state, git status, and current diff as source of truth.
- Keep `.agent/resume.md` compact, overwrite-only, and under 20 lines.
- Do not create long handoff logs or growing history files.
- Use `/rollover` to checkpoint context-heavy work.
- In a fresh session, use `/rehydrate` before continuing.
- Do not name any custom command `/resume` — that is a built-in Claude Code command.
